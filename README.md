# Workasana - task management and team collaboration tool
a task management and team collaboration tool where users can create projects, assign tasks to teams and owners, set deadlines, and organize work using tags. It supports authentication, dynamic filtering, URL-based queries, and reporting features to track task progress and team productivity. 

Built with React frontend, Express/Node backend, user authentication and MongoDB database.
